###Commands

  1.  source Scripts/activate
            Activate virtual env
  2.  tar zxvf dependencies.tar.gz
            Unpack dependencies
  3.  cd Dependencies
            Change Directory
  4.  pip install * -f ./ --no-index
            Install Packages
  5.  cd.. 
            change Directory
  6.  python main.py -c true
            Run main file 
  7.  deactivate
            quite virtualenv
