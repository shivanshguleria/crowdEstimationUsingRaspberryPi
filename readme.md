## Commands

  1.  ```bash
        source Scripts/activate 
      ```
  2.  ```bash
        tar zxvf dependencies.tar.gz
      ```
  3.  ```bash
        cd Dependencies
      ```
  4.  ```bash
        pip install * -f ./ --no-index
      ```
  5.  ```bash
        cd..
      ```
  6.  ```bash
        python main.py -c true
      ```
  7.  ```bash
        deactivate
      ```
